# Project setup

## Frontend
- Initialize React + Vite + TypeScript in /frontend
- Add Tailwind CSS
- Add react-i18next
- Add PWA support later when base app works
- Run with: npm run dev

## Backend
- Initialize FastAPI in /backend
- Add a basic health endpoint
- Keep a simple requirements file or pyproject.toml
- Run with uvicorn

## General
- Keep frontend and backend independently runnable
- Start with mock data if backend routes are incomplete
- Do not overengineer the setup
